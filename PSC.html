Welcome to Password Strength Checker

This application is designed to check the strength of your password which is very necessary as passwords are there to protect you personal data.
Enter the password
It is operated through Light 1
1.Weak Password = Red
2.Moderate Password = Blue
3.Strong Password = Green
4.Short Password = Yellow
5.Space In Password = Aurquoise Light
Password strength is a measure of the effectiveness of a password against guessing or brute-force attacks. This application tells you that a password with just alphabets is a Weak password adding some numbers make it Moderately hard to guess but to be completely secure one should add symbols to their password too.
